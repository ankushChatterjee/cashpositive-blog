# cashpositive-blog

# FRONT-END JAVASCRIPT ASSIGNMENT - BLOG

# Installation
To install all the dependendencies.
```
yarn add .
```
# Running
```
yarn run start
```
# Building
```
yarn run build
```

**Deployed URL**:- https://cashpositive-blog.firebaseapp.com/

# Technologies Used
1. ReactJS
2. Redux
3. React Router 
4. yarn
5. ParcelJS (for module loading and building)
6. Firebase (for deployment)
